# Enigma Machine Simulator

A TypeScript implementation of the historic **Enigma machine**, including configurable rotors, plugboard, and reflector components. This project allows encoding and decoding messages using classical rotor-based encryption.

---

## Features

- **Customizable Rotors:** Supports multiple rotors (I–V) with adjustable positions.
- **Plugboard Support:** Allows swapping pairs of letters before and after the rotor sequence.
- **Reflectors:** Implements standard reflectors (A, B, C) to map outputs back through the rotors.
- **Letter-by-Letter Encoding:** Handles encoding of single letters or full words.
- **Detailed Debug Logging:** Optional step-by-step logging for tracing the encryption process.
